Este projeto consiste em um gerenciador de tarefas desenvolvido em Python, executado via terminal (CLI). A aplicação permite ao usuário adicionar, listar, concluir e remover tarefas de forma simples e intuitiva.

As informações são armazenadas em um arquivo JSON, garantindo persistência dos dados mesmo após o encerramento do programa. O projeto foi criado com o objetivo de praticar conceitos fundamentais de programação, como estruturas condicionais, laços de repetição, funções, manipulação de listas e dicionários, além de leitura e escrita de arquivos.

Essa aplicação serve como base para projetos mais avançados, demonstrando organização de código e boas práticas no desenvolvimento com Python.
